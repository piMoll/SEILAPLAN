about=[DE]
    SEILAPLAN ermöglicht die einfache und effiziente Planung von Seillinien unter Berücksichtigung topografischer Gegebenheiten. Die implementierte Berechnungsmethode basiert auf der Methode von Zweifel (1960) und beschreibt die Kettenlinie, unter Annahme einer beidseitig fixierten Verankerung des Tragseils.
    Dieses QGIS-Plugin ist ein Hilfsmittel. Jede Lösung muss vor Umsetzung durch eine geschulte Fachperson geprüft werden. Jegliche Haftung wird im Rahmen der GNU General Public Licence Version 2 oder neuere ausgeschlossen (www.gnu.org/licenses).
    ***
    [EN]
    SEILAPLAN enables simple and efficient planning of cable roads, taking topographical conditions into account. The implemented calculation method is based on the method developed by Zweifel (1960) and describes the catenary, assuming that the skyline is anchored at both ends.
    This QGIS plugin is a tool. Every solution must be checked by a trained specialist before implementation. Any liability is excluded under the GNU General Public License Version 2 or later (www.gnu.org/licenses).

     Neue Features
     - Export von Geodaten im GeoPackage-Format (*.gpkg)
     - PDF-Export des Diagramms in den Papierformaten A4 und A3
     - Relative Speicherung der Pfade zu Höhendaten in der SEILAPLAN-Projektdatei, sodass Projekte und Höhendaten gemeinsam verschoben und wieder geöffnet werden können
     - Diverse Verbesserungen der Benutzeroberfläche und der Codequalität

     Fehlerbehebung
     - Karte wird korrekt auf Rasterlayer zentriert, auch wenn mehrere Raster in der Auswahlliste ausgewählt werden
     - Höhenlinien können nun auch aus Cloud-optimierten GeoTIFF-Rastern (COG) berechnet werden
     - Neue Versionen des SEILAPLAN-Plugins werden vom Plugin-Manager wieder erkannt und können aktualisiert werden

    ***
    [EN]

    New features
    - Export geospatial data in GeoPackage format (*.gpkg)
    - Export diagrams as PDF in A4 and A3 paper formats
    - Store elevation data paths as relative paths in SEILAPLAN project files, allowing projects and elevation data to be moved and reopened together
    - Various user interface and code quality improvements

    Fixes
    - The map is now correctly centered on raster layers, even when multiple raster layers are selected
    - Contour lines can now also be generated from Cloud Optimized GeoTIFF (COG) rasters
    - New SEILAPLAN plugin versions are now correctly detected by the QGIS Plugin Manager and can be updated

    Full changelog: https://github.com/piMoll/SEILAPLAN/blob/master/CHANGELOG.md
